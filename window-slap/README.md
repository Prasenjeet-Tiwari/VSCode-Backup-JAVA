# Window Slap App

A playful interactive application that responds to slaps with satisfying audio-visual feedback.

## Features

- Interactive "window" that responds to clicks/taps
- Multiple sound packs (Glass, Wood, Metal, Silly)
- Visual crack effects based on intensity
- Screen shake for heavy slaps
- Statistics tracking with achievements
- Responsive design

## Quick Start

### Development

```bash
# Install dependencies
cd backend && npm install

# Start server
npm run dev

# Open http://localhost:3000
```
